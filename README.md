Task 1: First Project Java Spring
A simple web application using Java, Spring Boot, and Thymeleaf.

Technologies
Java 21

Spring Boot

Thymeleaf

Maven

How to Run
Open in IDE.

Execute command:

Bash

./mvnw spring-boot:run
Server URL: http://localhost:8080

Endpoints
1. Root Page
   URL: http://localhost:8080/

Description: Simple text message.
